## Gist
Gist object

### Properties
* id - Identifier of a gist
 * Example: `"1"`
 * Type: string

### GET /gists
List the authenticated user's gists or if called anonymously, this will return all public gists

```
GET /gists HTTP/1.1
Content-Type: application/json
Host: api.github.com
```

```
HTTP/1.1 200
Content-Type: application/json

{
  "id": "1"
}
```

### POST /gists
Create a gist

```
POST /gists HTTP/1.1
Content-Type: application/json
Host: api.github.com

{
  "id": "1"
}
```

```
HTTP/1.1 201
Content-Type: application/json

{
  "id": "1"
}
```

### GET /gists/:id

```
Host: api.github.com
```
